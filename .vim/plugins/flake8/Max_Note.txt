To use Flake8 you need to do...

pip install flake8



To change whether we use python2-flake8 or python3-flake8, we need to change 

call s:DeclareOption('flake8_cmd', '', '"flake8"')
...to
call s:DeclareOption('flake8_cmd', '', '"python3-flake8"')

To do this, you must also install python3-flake8. I installed this using pip,
but it doesn't give you an executable called python3-flake8. So, I had to make
a new one!

localhost$ sudo vim /usr/bin/python3-flake8

---/usr/bin/python3-flake8---

#! /usr/bin/python3
__requires__ = 'flake8==2.1.0'
import sys
from pkg_resources import load_entry_point

if __name__ == "__main__":
    sys.exit(
        load_entry_point('flake8==2.1.0', 'console_scripts', 'flake8')()
    )

---END /usr/bin/python3-flake8---

localhost$ sudo chmod +x /usr/bin/python3-flake8
